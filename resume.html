Project-3: Garbage Level Indicator

Technologies Used: C++, Arduino IDE, Arduino Uno

In this Project we have build a hardware circuit which is capable of sensing the level of Garbage filled in the Dustbin and intimate respective Authorities based on the height of the garbage. The project requires one Arduino Uno, GSM module SIM900A, Ultrasonic sensor and two LED’s. The Sensor senses the distance and informs to Arduino Uno which helps to decides whether particular height Threshold is meet or not, if meet it sends an SMS alert to respective Authority. This project has a lot applications like reducing Environmental Pollution, Disciplined Timing in removal of Garbage, Helps to make the city clean
